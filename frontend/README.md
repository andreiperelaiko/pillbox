# Tabletnica

Приложение для управления приемом медикаментов с напоминаниями и уведомлениями опекунам.

## Установка

```bash
# Установка зависимостей
npm install
```

## Разработка

### Запуск dev сервера

```bash
npm run dev
```

Запускает фронтенд dev сервер (Vite) на порту 5173.

## Сборка для продакшн

### Базовая сборка

```bash
npm run build
```

Этот скрипт:
1. Проверяет TypeScript типы (`tsc`)
2. Собирает проект с оптимизацией (`vite build`)
3. Создает папку `dist/` с готовым к деплою кодом

### Полная сборка с проверками

```bash
npm run build:prod
```

Выполняет:
1. Форматирование кода (Prettier)
2. Проверку линтером (ESLint)
3. Сборку проекта

### Предпросмотр сборки

```bash
npm run preview
```

Запускает локальный сервер для проверки собранного проекта на `http://localhost:4173`

## Развертывание

Подробное руководство по развертыванию на сервер см. в [DEPLOY.md](./DEPLOY.md)

### Быстрый старт

1. Соберите проект:
```bash
npm run build
```

2. Загрузите содержимое папки `dist/` на ваш сервер

3. Настройте веб-сервер (Nginx, Apache и т.д.) для обслуживания статических файлов

4. Убедитесь, что API сервер доступен по адресу `http://193.106.150.7:8000/api/`

## Особенности

- **Внешний API**: Используется внешний API сервер `http://193.106.150.7:8000/api/`
- **SPA приложение**: React приложение с клиентской маршрутизацией
- **Redux Toolkit**: Управление состоянием через Redux Toolkit

## Структура проекта

```
tabletnica/
├── src/              # Фронтенд (React + TypeScript)
│   ├── api/          # API клиенты
│   ├── components/   # React компоненты
│   ├── pages/        # Страницы приложения
│   ├── store/        # Redux store и slices
│   └── utils/        # Утилиты
├── dist/             # Собранное приложение (создается после build)
├── nginx.conf        # Конфигурация Nginx для деплоя
├── .env.production   # Переменные окружения для продакшн
└── package.json      # Зависимости и скрипты
```

## Скрипты

- `npm run dev` - Запуск dev сервера
- `npm run build` - Сборка для продакшн
- `npm run build:prod` - Полная сборка с проверками
- `npm run preview` - Предпросмотр собранного проекта
- `npm run lint` - Проверка кода линтером
- `npm run format` - Форматирование кода
- `npm run format:check` - Проверка форматирования

## API Endpoints

Все API endpoints доступны через `/api`:

- `GET/POST/PATCH/DELETE /api/medications` - Медикаменты
- `GET/POST/PATCH/DELETE /api/intakes` - Приемы медикаментов
- `GET/POST/PATCH/DELETE /api/caregivers` - Опекуны
- `GET/PATCH /api/settings` - Настройки

API работает на внешнем сервере `http://193.106.150.7:8000/api/`

## Переменные окружения

- `VITE_API_BASE_URL` - Базовый URL API (по умолчанию: `http://193.106.150.7:8000/api/`)

Создайте файлы:
- `.env.development` - для разработки
- `.env.production` - для продакшн
